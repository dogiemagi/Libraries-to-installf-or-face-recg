Library Tools Installation Guide

1. Upgrade pip to the latest version

python3 -m pip install --upgrade pip

2. Install Required Python Libraries

Seaborn

pip install seaborn
pip show seaborn

Matplotlib

pip install matplotlib
pip show matplotlib

NumPy

pip install numpy
pip show numpy

Pandas

pip install pandas
pip show pandas

3. Install OpenCV

py -m pip install opencv-contrib-python

To verify the installation:

import cv2
print(cv2.__version__)

4. Uninstall NumPy (If Needed)

pip uninstall numpy

5. Install CMake

Download and install CMake from the official website.

cmake --version

6. Install dlib

Install via pip

pip install dlib

Alternative Method

If pip fails, download the appropriate .whl file from Dlib GitHub page and install it:

pip install filename.whl

7. Install Face Recognition

pip install face-recognition

Verify the installation:

import face_recognition
print(face_recognition.__version__)

8. Install Flask

pip install flask

Verify the installation:

import flask
print(flask.__version__)
